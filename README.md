# Column

## Contributing / Development

### git clone & npm install
```bash
$ git clone https://github.com/popodidi/Column
$ npm install
```

### Semantic UI

```bash
$ npm install gulp -g
$ cd semantic
$ gulp build
```

### Electron rebuild
```bash
$ npm run rebuild
```

### Debugging
```bash
$ npm run dev-server
$ npm run dev-electron
```

### Export Application
```bash
$ npm run pro-webpack
$ npm run pro-electron # check the output
$ npm run build-for-mac
```
